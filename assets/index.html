Sobre Mí

Soy un Desarrollador Web con 3 años de experiencia en la creación y soporte de aplicaciones y proyectos utilizando tecnologías como JavaScript, TypeScript, C# y algo de PHP. Poseo conocimientos en POO y funcional también en frameworks como React, Tailwind CSS y Bootstrap, así como en el consumo de APIs mediante Fetch y RESTful y la gestión de bases de datos SQL con MariaDB y MySQL, conocimientos básicos en SAP BTP y SAP Fiori.
