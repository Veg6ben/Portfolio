Mis Proyectos

Algunos de los proyectos que he desarrollado desplegados en vercel

Bienes Raices

El proyecto Bienes Raíces usa CSS Grid y Flexbox para el diseño responsivo, JavaScript para interactividad y manejo de datos, y tecnologías como HTML5, Sass, y despliegue en Vercel. Utiliza fuentes externas, iconos FontAwesome y buenas prácticas de organización de archivos.

Diseño Web

El proyecto utiliza HTML5 para la estructura, CSS3 (Grid y Flexbox) para el diseño responsivo, JavaScript para interactividad y manejo de mapas, fuentes externas, iconos FontAwesome y despliegue en Vercel

Api de clima

HTML5 para la estructura, CSS3 para el diseño responsivo y visual, y JavaScript para consumir una API de clima, procesar datos y mostrar el pronóstico en tiempo real

Festival musica

HTML5 para la estructura, CSS3 (incluyendo Grid y Flexbox) para el diseño visual y responsivo, y JavaScript para la interactividad y manejo de eventos. Presenta información sobre un festival musical, con secciones de artistas, horarios y compra de boletos

Pacientes veterinaria

proyecto está construido con HTML5 para la estructura de la interfaz, CSS3 para estilos y adaptación a dispositivos móviles, y JavaScript para la lógica de gestión de citas veterinarias, permitiendo crear, editar y eliminar registros. Utiliza LocalStorage para persistencia de datos en el navegador

Buscador de imagenes

Buscador de imágenes que utiliza la API de Pixabay. Permite a los usuarios buscar imágenes por palabra clave, mostrando resultados de forma dinámica con HTML5, CSS3 para el diseño responsivo y JavaScript para consumir la API y renderizar las imágenes en tiempo real.

Gasto semanal

Permite registrar y controlar gastos semanales. Utiliza HTML5 para la estructura, CSS3 para el diseño responsivo y JavaScript para gestionar el presupuesto, calcular gastos y mostrar el saldo restante dinámicamente

Buscador de coches

El sitio permite buscar y filtrar autos según diferentes criterios, mostrando los resultados de manera clara y ordenada. Los usuarios pueden explorar opciones disponibles y encontrar vehículos que se ajusten a sus preferencias
